# 统一检索接口使用说明

## Python检索示例

```python
import json
import os

# 产品知识库检索
with open('knowledge-base/product/product_index.json') as f:
    product_kb = json.load(f)

# 按关键词搜索
results = [doc for doc in product_kb['documents']
           if '安全' in doc.get('summary', '')]

# CISSP知识库检索
with open('knowledge-base/cissp/cissp_kb.json') as f:
    cissp_kb = json.load(f)

# 按领域检索
domain_docs = cissp_kb['domains']['安全与风险管理']['documents']
```

## 命令行检索示例 (jq)

```bash
# 搜索包含'漏洞'的文档
jq '.documents[] | select(.summary | contains("漏洞"))'
   knowledge-base/product/product_index.json
```
