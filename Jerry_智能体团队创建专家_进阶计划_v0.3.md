## 📦 Skill 层 - 核心工具链设计

### 1. 智能体生成器（agent-generator）
**功能：** 根据业务需求自动生成完整的智能体定义
**输入：** 行业、业务场景、核心目标
**输出：** 完整的Agent定义（角色+目标+背景+权限+技能集）
**脚本：** `scripts/generate_agent.py`

### 2. 团队架构生成器（team-architect）
**功能：** 自动设计多智能体团队架构
**输入：** 业务领域、复杂度、规模需求
**输出：** 完整的三层架构设计 + 通信协议 + 协作模式
**脚本：** `scripts/generate_team.py`

### 3. 技能打包器（skill-packager）
**功能：** 标准化技能创建与打包
**输入：** 技能需求描述
**输出：** 符合OpenClaw规范的Skill包
**脚本：** 基于官方skill-creator增强

### 4. 知识库构建器（knowledge-builder）
**功能：** 自动化构建结构化知识库
**输入：** 原始文档、数据
**输出：** 结构化知识库 + 向量索引
**脚本：** `scripts/build_knowledge.py`

### 5. 质量验证器（quality-validator）
**功能：** 智能体团队质量验证
**检查项：** 安全闸门、权限边界、性能指标、成本预估
**脚本：** `scripts/validate_agent.py`

## 🔧 标准工作流程（7步智能体团队创建SOP

```
第1步：需求分析 ← 用户输入业务需求
    ↓
第2步：行业匹配 ← 从模板库匹配最接近的行业模板
    ↓
第3步：架构设计 ← 生成三层团队架构
    ↓
第4步：角色定义 ← 每个Agent的专业化定义
    ↓
第5步：技能配置 ← 为每个Agent配置技能集
    ↓
第6步：质量验证 ← 安全+性能+成本三重验证
    ↓
第7步：交付上线 ← 完整交付 + 运维手册
```
