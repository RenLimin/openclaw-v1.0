## 🧬 进化模式库说明

### 什么是进化模式？
进化模式是从大量真实错误和最佳实践中抽象提炼的、可复用的通用优化模式。

### 模式分类 (v1.0 共 6 个模式)

| 分类 | 模式数量 | 说明 |
|------|---------|------|
| 🔴 错误预防类 | 2个 | 防止同类错误再次发生的模式 |
| 🟡 流程改进类 | 2个 | 优化工作流程的标准模式 |
| 🟢 架构最佳实践类 | 2个 | 智能体团队架构设计模式 |
| 🔵 团队创建类 | 0个 | 待添加 |

## 🔍 模式查询工具

使用 `evolution_engine.py` 可以查询和应用进化模式：

## 💡 贡献指南

每一次错误和成功经验都是宝贵的知识资产：

1. 发生错误 → 完整记录到 ERRORS.md
2. 提炼根因 → 执行 5WHY 分析
3. 抽象模式 → 如果具有通用性，考虑加入进化模式库
4. 验证效果 → 优化后验证效果并闭环
