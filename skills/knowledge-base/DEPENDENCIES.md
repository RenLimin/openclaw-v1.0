# 合同法规知识库 - 依赖说明

---

## 📦 Python 依赖包

| 包名 | 版本要求 | 用途 | 必选 |
|------|---------|------|------|
| `pyyaml` | ≥ 6.0 | YAML 法规数据文件解析 | ✅ 是 |
| `fuzzywuzzy` | ≥ 0.18.0 | 关键词模糊匹配 | ⭕ 否（可选增强） |
| `python-Levenshtein` | ≥ 0.23.0 | 字符串相似度计算加速 | ⭕ 否（可选增强） |
| `jieba` | ≥ 0.42.1 | 中文分词用于关键词匹配 | ⭕ 否（可选增强） |

---

## 🌐 系统依赖

| 依赖 | 说明 |
|------|------|
| **Python 3.8+** | 最低版本要求 |

---

## 🚀 安装命令

### 基础安装

```bash
pip install pyyaml
```

### 完整安装（含模糊匹配增强）

```bash
pip install pyyaml fuzzywuzzy python-Levenshtein jieba
```
