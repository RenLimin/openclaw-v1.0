## 🔄 文档维护规范

### ✅ 新增文档时
1. 按上述分类放入对应目录
2. 在本文档的「完整文档体系结构」中更新
3. 在「文档依赖关系图」中标明与其他文档的关系

### ✅ 更新文档时
1. 同步更新所有引用了它的文档
2. 重大更新需要在「版本演进里程碑」中记录
3. 旧版本放入 `_archive/` 目录归档

### ✅ 归档原则
- 小版本迭代（v4.0 → v4.1）：旧版本直接覆盖或删除
- 大版本重构（v3.x → v4.x）：旧版本放入 `_archive/` 保留参考
