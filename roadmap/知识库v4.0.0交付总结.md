## 💡 关键技术决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 主数据库 | SQLite 3.x | 零依赖、单文件、足够性能、运维简单 |
| 全文检索 | FTS5 (内置) | 避免引入 ES/Solr 等重型依赖 |
| 向量索引 | FAISS | Facebook 开源、性能优异、本地部署 |
| 增量检测 | SHA256 哈希 | 精确可靠，不依赖文件系统时间 |
| 分块策略 | 语义分块优先 | 比固定大小分块检索质量更高 |
| 部署模式 | 本地独立部署 | 可脱离大模型独立运行，成本最低 |
