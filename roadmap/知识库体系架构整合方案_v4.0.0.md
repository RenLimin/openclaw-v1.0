## 📋 整合原则

### 1. **知识库 = 全局共享基础设施**
```
✅ 所有 Agent 共享同一个知识库，避免重复建设
✅ 知识一次入库，全团队可用
✅ 统一的知识质量标准和审核流程
✅ 知识版本全局一致，不会出现偏差
```

### 2. **能力分层，解耦设计**
```
┌─────────────────────────────────────────┐
│    智能体层（业务逻辑、推理、决策）     │
└───────────────┬─────────────────────────┘
                │ 调用
┌───────────────▼─────────────────────────┐
│    知识库层（存储、检索、关联、更新）   │
└─────────────────────────────────────────┘
```

### 3. **统一 API，多语言兼容**
```python

# 命令行调用
kb_manager.py search "关键词"
```

### 方式二：对话过程中的 RAG 检索增强
```python
# 在对话循环中，每个用户问题都先检索知识库

def handle_user_query(agent, user_question: str):
    # Step 1: 检索相关知识
    knowledge_results = agent.kb.hybrid_search(
        query=user_question,
        top_k=10
    )
    
    # Step 2: 构建增强 Prompt
    augmented_prompt = f"""
    请基于以下知识库内容回答用户问题：
    
    【知识库检索结果】
    {format_results(knowledge_results)}
    
    【用户问题】
    {user_question}
    
    要求：
    1. 优先使用知识库中的信息回答
    2. 不确定的内容明确说明
    3. 引用的知识来源请标注
    """
    
    # Step 3: 生成回答
    return agent.generate(augmented_prompt)
```

```
├─────────────────────────────────────────────────────┤

└─────────────────────────────────────────────────────┘
```

### 安全闸门集成
```python
# 在 security_guard.py 中新增知识库操作检查

def check_knowledge_operation(agent_name: str, operation: str, **kwargs):
    """知识库操作安全检查"""
    
    RULES = {
        "add_document": ["知识管理员"],  # 只有知识管理员可以新增
        "update_document": ["知识管理员"],
        "delete_document": ["全局管理员"],  # 删除需要最高权限
        "search": ["*"],  # 所有人可以检索
        "export": ["知识管理员", "全局管理员"]
    }
    
    # 验证角色权限
    if not has_permission(agent_name, RULES[operation]):
        raise SecurityError(f"Agent {agent_name} 无权限执行 {operation}")
    
    # 敏感数据脱敏
    if operation in ["search", "export"]:
        kwargs["content"] = mask_sensitive_data(kwargs["content"])
    
    # 审计日志
    log_audit(agent_name, operation, kwargs)
    
    return True
```
