# 知识库体系结构说明文档

## 体系架构总览

```
knowledge-base/
├── ones/          # (7) ONES系统知识库
├── product/       # (8) 产品知识库
└── cissp/         # (9) CISSP专业知识库
```

## 各知识库详情

### ONES系统知识库 (ones/)
- 数据源: https://ones.bangcle.com/
- 包含: 页面内容 + 附件 + 链接 + 问题日志

### 产品知识库 (product/)
- 数据源: Bangcle Product Material 目录
- 索引字段: 文件名、路径、类型、摘要、关键词、元数据
- 支持格式: Word/Excel/PDF/PowerPoint/Markdown/Text

### CISSP专业知识库 (cissp/)
- 数据源: CISSP学习目录
- 按八大知识领域分类索引
- 保留原始文件只读，不移动不修改
- 章节索引支持按知识点检索

## CISSP八大知识领域
1. 安全与风险管理 (Domain 1)
2. 资产安全 (Domain 2)
3. 安全架构与工程 (Domain 3)
4. 通信与网络安全 (Domain 4)
5. 身份与访问管理 (Domain 5)
6. 安全评估与测试 (Domain 6)
7. 安全运营 (Domain 7)
8. 软件开发安全 (Domain 8)
