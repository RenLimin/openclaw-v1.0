# 产品知识库查询技能

## 概述

梆梆安全产品和服务知识库，整合了6条产品线的1763个产品文档，支持多维度智能查询。

## 产品线覆盖

| 产品线 | 主要产品 |
|--------|---------|
| **安全监测线** | 移动应用安全监测、API安全平台、全渠道应用安全监测、端到端安全监测 |
| **安全检测线** | 应用安全测评、移动应用合规、固件安全检测、SDK安全检测 |
| **安全保护线** | Android/iOS/H5/小程序/SDK/鸿蒙/物联网应用加固、安全键盘、防界面劫持、密钥白盒、通信协议保护 |
| **内容安全线** | 内容安全检测、内容审核、舆情监测 |
| **安全服务线** | 安全评估、渗透测试、等保咨询、应急响应、安全培训、合规咨询 |
| **物联网服务线** | 汽车信息安全测试、物联网安全测评、车载安全检测 |

## 使用方法

### 🔄 OA知识库自动同步更新
```bash
# 查看同步状态
python oa_sync.py status

# 只同步知识文档
python oa_sync.py doc

# 只同步产品清单
python oa_sync.py product

# 完整同步（文档 + 产品）
python oa_sync.py full
```

**同步功能说明：**
1. **知识文档同步**：OA路径「应用中心 - 知识 - 查询文档 - 产品中心」，获取所有 2026-01-01 及之后创建/更新的文档
2. **产品清单同步**：OA路径「产品研发管理系统 - 产品基本信息管理 - 梆梆_产品价格查询」，获取全部记录作为标准产品服务清单
3. **自动记录更新时间**：作为下次增量更新的截止日期，实现增量同步

### 3. 查询产品部署安装说明
```bash
python main.py deploy "全渠道应用安全监测"
```

```bash
python main.py step

```
