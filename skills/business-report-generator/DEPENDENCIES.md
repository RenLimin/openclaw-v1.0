# 经营报告生成技能 - 依赖说明

---

## 📦 Python 依赖包

| 包名 | 版本要求 | 用途 | 必选 |
|------|---------|------|------|
| `python-docx` | ≥ 0.8.11 | Word 文档生成 | ✅ 是 |
| `pandas` | ≥ 2.0.0 | 数据处理和分析 | ✅ 是 |
| `openpyxl` | ≥ 3.1.0 | Excel 文件读写 | ✅ 是 |
| `matplotlib` | ≥ 3.7.0 | 图表生成 | ✅ 是 |
| `seaborn` | ≥ 0.12.0 | 高级统计图表 | ⭕ 否 |
| `jinja2` | ≥ 3.1.0 | 模板渲染引擎 | ✅ 是 |
| `markdown` | ≥ 3.4.0 | Markdown 解析 | ⭕ 否 |
| `python-dotenv` | ≥ 1.0.0 | 环境变量加载 | ⭕ 否 |
| `numpy` | ≥ 1.24.0 | 数值计算 | ✅ 是 |

---

## 🌐 系统依赖

| 依赖 | 说明 |
|------|------|
| **Python 3.8+** | 最低版本要求 |
| **中文字体支持** | 生成的图表需正确显示中文 |
| **足够内存** | 处理大数据时建议 ≥ 4GB |

---

## 🚀 安装命令

### 基础安装

```bash
pip install python-docx pandas openpyxl matplotlib jinja2 numpy
```

### 完整安装（含可选依赖）

```bash
pip install python-docx pandas openpyxl matplotlib seaborn jinja2 markdown python-dotenv numpy
```

---

## 🔗 依赖技能

| 技能名称 | 用途 | 必须 |
|---------|------|------|
| `summarize-pro` | 报告摘要智能生成 | ⭕ 否（可选增强功能） |
| `excel-xlsx` | 高级 Excel 处理 | ⭕ 否（可选增强功能） |

---

## 🔐 安全依赖

| 项目 | 说明 |
|------|------|
| **本地处理** | 所有经营数据必须本地处理，禁止上传到第三方模型服务 |
| **数据脱敏** | 支持敏感数据（如人员薪资、具体金额）自动脱敏 |
