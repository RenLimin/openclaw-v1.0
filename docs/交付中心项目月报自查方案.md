# 交付中心项目月报自查方案

## 1. 检查项清单

| 检查项 | 检查内容 | 合格标准 | 检查方式 |
|--------|---------|---------|---------|
| 1.1 A1报表日期 | A1单元格日期格式 | 格式正确，为当月最后一天 | 人工查看Excel |
| 1.2 表头完整性 | 所有表头列是否齐全 | 与模板一致 | 人工查看 |
| 1.3 表头格式 | 表头样式 | 蓝底白字、居中和粗体 | 人工查看 |
| | | | |
| 2.1 公式列完整性 | 所有公式列是否保留 | 与模板公式位置一致 | Excel查看公式 |
| 2.2 公式计算正确性 | 公式计算结果是否正确 | 抽样手工验证3-5行 | 人工验证 |
| 2.3 跨表引用正确性 | $列$行引用是否正确 | 与模板引用位置一致 | 人工查看 |
| | | | |
| 3.1 数据行数完整性 | 数据是否完整导入 | 与CSV源文件行数一致 | 对比CSV文件 |
| 3.2 数据内容正确性 | 抽样数据是否正确 | 内容一致无乱码无缺失 | 抽样对比 |
| 3.3 日期格式统一 | 日期列格式是否统一 | YYYY-MM-DD格式 | 人工查看 |
| | | | |
| 4.1 表格边框 | 是否全表有标准边框 | 四周边框完整 | 人工查看 |
| 4.2 字体格式 | 字体统一 | 统一微软雅黑11号 | 人工查看 |
| 4.3 列宽自适应 | 列宽是否合适 | 内容完整显示不挤 | 人工查看 |
| | | | |
| 5.1 统计结果正确性 | 统计表数字是否合理 | 趋势合理无异常大数字 | 人工核对 |
| 5.2 空值处理 | 统计空行是否正确 | 0或空值合理 | 人工查看 |
| | | | |
| 6.1 文件完整性 | 所有工作表是否齐全 | 15个sheet完整 | 人工查看 |
| 6.2 文件大小合理性 | 文件大小是否正常 | 与模板大小差异不超2倍 | 文件属性查看 |

## 2. 自查流程

```
第1步：打开Excel → 检查15个工作表是否齐全
       预计耗时：1分钟

第2步：逐个检查5个核心sheet
   - 签约
   - POC&提前实施
   - 异常项目
   - 确收交接
   - 验收交接
   每个sheet检查：
   2.1 A1日期是否为当月最后一天
   2.2 表头列是否齐全，样式是否正确
   2.3 数据行数是否与源数据一致
   2.4 公式列是否有计算结果
   预计耗时：每个sheet 1分钟，共5分钟

第3步：抽样验证公式结果
   - 每个sheet随机选3-5行，手算公式是否正确
   - 检查跨表引用是否正常
   预计耗时：5分钟

第4步：检查统计表和图表
   - 各统计表数字是否合理
   - 图表是否正常显示
   预计耗时：3分钟

第5步：整体格式检查
   - 边框、字体、列宽
   - 文件大小是否合理
   预计耗时：2分钟

总计：16分钟
```

## 3. 验收标准

✅ 全部检查项通过 → 交付合格
⚠️ 3项以下轻微问题（不影响数据正确性）→ 记录后可交付
❌ 数据错误/公式错误/核心表缺失 → 必须修复后重新自查

## 4. 常见问题清单

| 问题 | 原因 | 修复方法 |
|------|------|---------|
| A1日期不对 | 硬编码了日期 | 动态计算月份最后一天 |
| 公式列空白 | data_only=True打开或写入覆盖了 | 用data_only=False打开，不写公式列 |
| 格式混乱 | 未继承模板格式 | 先复制模板再写数据 |
| 部分行边框缺失 | 只写了数据没应用样式 | 循环整表加边框 |
| 统计结果异常大 | 空值或列错位导致 | 检查空值处理，检查数据写入列对齐 |
