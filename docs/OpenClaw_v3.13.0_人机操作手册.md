## 🟢 P0 工具级：系统快速安装（30分钟）

### 🎯 目标
从零开始搭建可用的 OpenClaw 智能体系统，5分钟跑起来，30分钟完成配置。

---

### 👨‍💻 **人操作步骤**

#### 步骤 1：环境准备
**执行人员**：系统管理员
**预计时间**：10分钟

| 检查项 | 要求 | 验证命令 |
|--------|------|---------|
| 操作系统 | macOS 12+ / Linux (Ubuntu 20.04+) | `uname -a` |
| Python | 3.9+ | `python3 --version` |
| Node.js | 18+ | `node --version` |
| Git | 最新版 | `git --version` |
| 磁盘空间 | ≥ 5GB | `df -h` |
| 内存 | ≥ 8GB | `free -h` (Linux) / `top -l 1` (macOS) |

#### 步骤 2：安装 OpenClaw
**执行人员**：系统管理员
**预计时间**：5分钟

```bash
# 1. 安装 OpenClaw CLI
npm install -g openclaw

# 2. 验证安装
openclaw --version

# 3. 初始化工作空间
mkdir -p ~/.openclaw/workspace
cd ~/.openclaw/workspace

# 4. 启动 Gateway
openclaw gateway start
```

#### 步骤 3：验证基础服务
**执行人员**：系统管理员
**预计时间**：5分钟

```bash
# 检查 Gateway 状态
openclaw gateway status

# 浏览器访问：http://127.0.0.1:18789
# 看到 OpenClaw Dashboard 即为成功
```

---

### 🤖 **机操作步骤**（自动执行）

#### 步骤 4：核心配置初始化
**触发条件**：Gateway 启动成功后
**执行主体**：系统自动执行

```bash

cd ~/.openclaw/workspace

# 2. 创建必要目录结构（如果不存在）
mkdir -p core-config memory logs temp

# 3. 复制默认配置模板

```

#### 步骤 5：Python 依赖安装
**触发条件**：配置初始化完成
**执行主体**：系统自动执行

```bash

pip3 install -r requirements.txt

# 验证关键库
python3 -c "import playwright; import openpyxl; import pandas; print('✅ 依赖安装成功')"
```

## 🔧 常见问题排查

### ❓ 问题 1：Gateway 启动失败
```bash
# 排查步骤：
1. 检查端口占用：lsof -i :18789
2. 查看日志：openclaw gateway logs
3. 重启服务：openclaw gateway restart
```

### ❓ 问题 2：技能调用失败
```bash
# 排查步骤：
1. 检查技能是否在 TOOLS.md 中启用
2. 检查依赖是否安装：pip3 install -r skills/xxx/requirements.txt
3. 查看详细日志：tail -f logs/skill-execution.log
4. 运行技能自检：pytest tests/test_xxx.py -v
```

### ❓ 问题 3：智能体回复质量差
```bash
# 排查步骤：
1. 检查是否幻觉：要求智能体提供引用来源
2. 检查知识库：是否导入了正确的领域知识
3. 检查 Prompt：是否清晰、具体、有约束
4. 降低温度：temperature 从 0.7 降到 0.3
5. 增加交叉验证：双 Agent 交叉验证输出
```

### ❓ 问题 4：大模型成本超支
