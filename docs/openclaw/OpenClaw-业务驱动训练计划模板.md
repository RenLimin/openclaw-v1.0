### 6.1 通用验收要求（所有技能必须满足）
- [ ] 所有密码/凭证通过 Keychain 存储，无明文硬编码

- [ ] 完整的 DEPENDENCIES.md（依赖清单+安装命令）
- [ ] 可重复运行的脚本（无绝对路径硬编码）
- [ ] 错误处理与重试机制
- [ ] 配置驱动（规则外置YAML，不硬编码）
